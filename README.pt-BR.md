## 🏪 O que estou construindo

### [EasyPDV](https://www.easypdv.com.br) — mercados autônomos, de ponta a ponta

Lojas de autoatendimento para condomínios e empresas: totem de checkout inteligente, ERP na nuvem, estoque, vendas e faturamento em tempo real, pagamentos por PIX · cartão · NFC — sem precisar de funcionários. Em produção, vendido como SaaS.

| Camada | O que está no ar |
|-------:|:-----------------|
| 📱 **Mobile** | 4 apps Android em Kotlin — PDV, autoatendimento touch (terminais Sunmi), apps do operador e do mercado |
| ⚙️ **Backend** | Java · Spring — APIs modulares & microsserviços (gateway, auth, service discovery) |
| 🖥️ **Web** | Painéis admin em TypeScript · sites em Astro · docs para clientes e devs |
| 🐍 **Dados & bots** | Crawlers em Python — enriquecimento de catálogo GTIN, automação de conteúdo e preços |
| 🚀 **Infra** | CI/CD & registry self-hosted — Docker · Harbor · Dokploy |
| 🔌 **Hardware** | Terminais de pagamento Sunmi · etiquetas eletrônicas BLE · integração com controle de acesso |

### 🧺 Despensinha

Linha de despensas inteligentes do mesmo ecossistema — micro-mercados para espaços menores: etiquetas IoT, microsserviços Java, storefronts TypeScript.

### 🔬 [Uniolab](https://github.com/Uniolab)

Meu estúdio e marca guarda-chuva — onde moram os produtos, as automações e os experimentos.
