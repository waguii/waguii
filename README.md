## 🏪 What I'm building

### [EasyPDV](https://www.easypdv.com.br) — autonomous mini-markets, end to end

Self-service stores for condos and companies: smart checkout kiosk, cloud ERP, real-time stock, sales and revenue, PIX · card · NFC payments — no staff required. In production, sold as SaaS.

| Layer | What's live |
|------:|:------------|
| 📱 **Mobile** | 4 Kotlin Android apps — POS, self-checkout touch (Sunmi terminals), operator & market apps |
| ⚙️ **Backend** | Java · Spring — modular APIs & microservices (gateway, auth, service discovery) |
| 🖥️ **Web** | TypeScript admin panels · Astro websites · client & developer docs |
| 🐍 **Data & bots** | Python crawlers — GTIN/product catalog enrichment, content & pricing automation |
| 🚀 **Infra** | Self-hosted CI/CD & registry — Docker · Harbor · Dokploy |
| 🔌 **Hardware** | Sunmi payment terminals · BLE electronic shelf tags · access control integration |

### 🧺 Despensinha

Smart-pantry line of the same ecosystem — micro-markets for smaller spaces: IoT price tags, Java microservices, TypeScript storefronts.

### 🔬 [Uniolab](https://github.com/Uniolab)

My studio & umbrella brand — where the products, automations and experiments live.
